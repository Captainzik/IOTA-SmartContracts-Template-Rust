# IOTA-SmartContracts-Template

A simple template used to start developing your own smart contracts for ISCP (IOTA Smart Contract Protocol) in Rust and write unit tests in Go. 

### Requirements
- Rust
- Wasm-pack
- Go
- Gcc (or equivalent for Windows)
